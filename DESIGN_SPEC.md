# 京都ファミリーウェディング 旅のしおり — デザイン構成書

## 概要

2026年11月28〜29日、広島→京都の家族旅行＋結婚式のためのデジタル招待状＆旅行ガイド。
タブで4ページに分かれたSPA構成。**スマホ閲覧が主**。

---

## デザインの方向性

- POPEYE Magazine 風のエディトリアルデザイン
- **日本語がメイン**、英語はあしらい（小さいイタリック体で添える程度）
- 紙のような質感: クリーム背景、太い黒ボーダー(3px)、オフセット影（box-shadow）
- カードは少し傾いている（rotate -1〜3deg）
- 各タブが雑誌の1ページのように独立した世界観を持つ

## インタラクション・アニメーション

| 要素 | アニメーション |
|---|---|
| スクロール reveal | 各要素がビューポートに入ると、下40pxから浮き上がり＋回転が正位置に戻る（0.65s） |
| タブ切替 | フェードイン＋上方向スライド（0.22s） |
| カードホバー | 浮き上がり（-4px）＋影が深くなる |
| 写真ホバー | 微ズームイン（1.06倍）＋彩度UP |
| スポットカードホバー | 浮き上がり（-5px）＋影が深くなる |
| カウントダウン数字 | ゆるく脈動するpulse（2秒周期、各数字に0.3sずつディレイ） |
| タブ背景パターン | ゆっくりドリフト移動（40〜60秒で1周期） |
| 浮遊装飾シェイプ | 各タブの背景にぼんやりした円形グラデーションがふわふわ上下（8〜12秒周期） |
| ページ番号ウォーターマーク | 各タブ右上に大きく薄い「01」〜「04」（静止、装飾用） |
| タイムラインアイテム | ホバーでうっすら暖色ハイライト |
| ボタン全般 | ホバーで影の方向に移動（押し込み感） |
