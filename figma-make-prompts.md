# Figma Make プロンプト集
## 京都ファミリーウェディング 旅のしおり

タブごとに4本のプロンプトに分割。毎回、冒頭に**共通スタイル定義**を貼ってから使う。

---

## 共通スタイル定義（全プロンプトの冒頭に貼る）

```
POPEYE Magazineのようなエディトリアル雑誌スタイルのモバイルUIデザイン。

カラーパレット：
  - paper  #f6f1e8（背景・カード）
  - ink    #111111（テキスト・ボーダー・影）
  - muted  #52493f（補足テキスト）
  - accent #d45b35（CTAボタン）
  - 黄色バッジ #f1c76d
  - ブルー #7fa7d8
  - グリーン #7a9b6e

フォント：
  - Noto Sans JP Black（ウェイト900）：大見出し
  - Noto Sans JP（ウェイト700/400）：本文・小見出し
  - Playfair Display Italic：英語の装飾テキスト
  - VT323（モノスペース）：数字表示

デザインルール：
  - 背景はクリーム色の紙質感
  - カードには太い黒ボーダー（3px）とオフセット影（4〜6px）
  - カードは少しだけ傾いている（−2°〜+3°）
  - グラデーションはカードに使わない
  - テキストは日本語メイン、英語は小さいイタリック体でアクセントとして添える
```

```
【セクション1：マストヘッドカード】

  左カラム：

  左：招待カード（クリーム背景、左に−1°傾く）

    - 見出し「みなさまへ、京都へようこそ」
    - 本文3段落（招待の挨拶文）
    - 署名「新郎新婦より」
    - 区切り線

黒背景のカード、右に+2°傾く。

  - サブ「挙式日時：2026.11.29 10:30 / アトールテラス鴨川」

```

## タブ4：出欠回答

```
モバイル画面、POPEYE Magazineのようなエディトリアル雑誌スタイル。

【背景】
朱色テーマ。温かみのあるオレンジ #fdf0ec を背景色に、斜めの菱形格子模様を重ねる。

【固定ヘッダー：タブナビゲーション】

【セクション1：2カラムレイアウト】
  左：案内カード（クリーム背景、左に−1°傾く）
    - 黄色バッジ「RSVP / 出欠回答」
    - 見出し「ご回答を、お願いします」
    - 案内文2段落

      「ご家族ごとにまとめてご回答ください。」

  右：フォームカード（クリーム背景、右に+1°傾く）
    すべての入力欄に太い黒ボーダー、クリーム背景。

    1. 代表者 ——— テキスト入力欄
    2. 連絡先 ——— テキスト入力欄
    3. 参加メンバー ——— セクション見出し＋メンバーカード
       「＋ メンバーを追加」ボタン（破線の黒ボーダー）
       メンバーカード（例）：
         - お名前：テキスト入力欄
         - 出欠：2択ラジオカード 【出席】【欠席】
           （選択済み：黒背景・白文字）
         - アレルギー：2択ラジオカード 【あり】【なし】
         - 「削除」小さいテキストリンク
    4. ひとことメモ ——— テキストエリア（3行）
    5. バス・宿泊メモ ——— テキストエリア（3行）
    6. 送信ボタン（全幅）
       朱色 #d45b35 背景、白文字「Send RSVP / 回答を送信」
```

## 使い方

1. 各プロンプトの**冒頭に「共通スタイル定義」を貼り付けてから**送る
2. **1タブずつ生成**して、気に入ったら次のタブへ進む
3. 生成後はFigmaで フォント・カラー・写真を手動で微調整する
4. タブ間でデザインがブレた場合は「前のタブのデザインに合わせて修正して」と追加指示する
