# RSVP回答をGoogleスプレッドシートに保存する手順

RSVPフォームの回答を、自分のGoogleスプレッドシートに自動で貯めるための設定です。
プログラミングは不要。**コピペとボタン操作だけ**でできます。所要5〜10分。

---

## 1. 保存先のスプレッドシートを作る

1. [Googleスプレッドシート](https://sheets.google.com) で新しいシートを1枚作る
2. 名前は何でもOK（例:「結婚式 出欠回答」）

## 2. Apps Script を開く

1. そのスプレッドシートのメニュー **「拡張機能」→「Apps Script」** を開く
2. 最初から入っているコードを**全部消す**
3. このリポジトリの [`gas/Code.gs`](gas/Code.gs) の中身を**全部コピーして貼り付ける**
4. 💾（保存）アイコンで保存

## 3. ウェブアプリとしてデプロイ（公開）

1. 右上の **「デプロイ」→「新しいデプロイ」**
2. 歯車⚙️ →「ウェブアプリ」を選択
3. 設定:
   - **実行するユーザー**: `自分`
   - **アクセスできるユーザー**: `全員`
4. 「デプロイ」を押す
5. 初回は **権限の承認** を求められる → 自分のGoogleアカウントで承認
   （「このアプリは確認されていません」と出たら「詳細」→「（安全でないページ）に移動」で進めてOK。自分が書いたスクリプトなので問題ありません）
6. 表示される **ウェブアプリ URL**（`https://script.google.com/macros/s/.../exec`）を**コピー**

## 4. エディターに貼り付ける

1. エディター（`/index/admin`）を開く
2. **RSVP（出欠フォーム）ブロック**を選択
3. 右の設定パネルの **「送信先URL（GAS）」** に、さっきコピーしたURLを貼り付け
4. 保存

これで完了！フロントのフォームから送信すると、スプレッドシートに1行ずつ追記されます。

---

## 動作確認

- フロント（`index.html`）でフォームを実際に送信してみる
- スプレッドシートに行が増えれば成功 🎉
- 「送信先URL」が空のままだと、フォームは「※ プレビューのため送信できません」と表示します（安全のための仕様）

## 保存される項目

受信日時 / 挙式 / 披露宴 / ゲスト区分 / 姓 / 名 / ローマ字 / 間柄 / 郵便番号 / 住所 / 電話 / メール / アレルギー / アレルギー内容 / **同席者人数** / **同席者（名前と出欠）** / メッセージ / 送迎バス

## フォームの項目を変えたとき

エディターでフォーム項目を増減しても、GAS側は送られてきた項目を自動で拾います。
ただし**列の見出しと並び**を厳密に合わせたい場合は、`gas/Code.gs` の `headers` と `row` の配列を同じ順番で編集してください。

## 注意（送信の仕組み）

静的サイトから外部に直接保存はできないため、GASを「中継サーバー」として使っています。
GASは無料枠で十分動きます（1日あたりの実行回数に上限はありますが、結婚式の出欠程度なら問題ありません）。
